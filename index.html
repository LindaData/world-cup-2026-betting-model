2026 World Cup Betting Model

A reproducible RStudio, Python, and SQL project for collecting World Cup data, building model-ready features, and testing transparent prediction baselines.

Current Baseline

The first model predicts team goals from historical international match data using pre-match Elo, opponent Elo, expected result, listed-home status, and neutral-site status. It is a simple OLS baseline, not the final betting model.

Workflow

The project is designed for RStudio. Python pulls data, DuckDB stores model-ready tables, and R scripts fit models and render reports.

Main entry points: world-cup-betting-model.Rproj, R/10_fit_linear_goals_model.R, and R/12_render_reports.R.

Limitations

This is a research project, not betting advice. The current model is intentionally simple. The next useful additions are odds snapshots, confirmed lineups, injuries, player minutes, and richer player-match statistics.

Nearest-neighbor comparison model for goals and match results

1 Purpose

The KNN model is a similarity model. For a team in a match, it finds historical team-match rows that look most similar, then summarizes what happened in those nearby games.

This gives the project a useful contrast:

  • OLS explains average goal changes with coefficients.
  • Ordinal logistic explains ordered win/draw/loss probabilities.
  • KNN asks: “What happened when past teams looked like this team in this matchup?”

2 How It Works

KNN stands for K-Nearest Neighbors.

For each test row, the model:

  1. Standardizes the model inputs so all features are on comparable scales.
  2. Finds the 25 closest historical team-match rows.
  3. Predicts goals by averaging the goals scored by those neighbors.
  4. Predicts loss, draw, and win probabilities by counting neighbor outcomes.

5 Interpretation

KNN is easy to explain because the prediction comes from comparable historical cases. That is also its limitation.

The model depends heavily on:

  • The features used to define similarity.
  • The number of neighbors.
  • Whether older matches should count the same as recent matches.
  • Whether rare outcomes, especially draws, are represented well enough in the neighbor set.

KNN is strongest as a comparison model. If it beats the simpler regression models, the feature set probably contains useful nonlinear patterns. If it loses, the added flexibility is not earning its keep.

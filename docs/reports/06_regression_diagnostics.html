Collinearity, residual checks, and count-model fit

1 Purpose

This report checks whether the first regression models are behaving well enough to interpret.

The checks focus on three practical questions:

  • Are the predictors repeating the same information?
  • Do residuals show patterns the model missed?
  • Is a count model a better shape for goals than simple linear regression?

4 Collinearity

Collinearity means two or more predictors are carrying nearly the same signal. It does not always ruin predictions, but it can make individual coefficients unstable.

VIF means variance inflation factor. A VIF near 1 is clean. Values above 5 deserve attention. Values above 10 are usually too high for coefficient interpretation.

5 Residual Checks

A residual is:

\[ \text{Residual} = \text{Observed Goals} - \text{Predicted Goals} \]

For OLS, residuals should not show strong structure after the model has done its work.

6 Poisson Checks

Poisson regression is a natural model for goals because goals are counts. The main issue is overdispersion.

Overdispersion means the observed goal counts vary more than a basic Poisson model expects. If this is large, the next goals model should consider a negative-binomial or paired-goal approach.

7 Interpretation

These diagnostics are the gate before model comparison:

  • If VIF is low, coefficient directions are easier to trust.
  • If Durbin-Watson is close to 2, residuals are not strongly sequential in the ordered rows.
  • If heteroskedasticity appears, OLS standard errors should be treated carefully.
  • If Poisson overdispersion appears, expected-goal predictions may still be useful, but uncertainty intervals need a wider model.

2026 match scores and win/draw/loss probabilities from local models

1 Purpose

This report scores the 2026 World Cup fixture list with the current local models.

Each fixture receives:

  • OLS expected goals for the home and away team.
  • Poisson expected goals for the home and away team.
  • Ordinal win, draw, and loss probabilities from the home teamâ€™s perspective.
  • Weather context when venue-level weather summaries are available.

5 Probability View

Show code 
if (nrow(predictions) > 0 && requireNamespace("ggplot2", quietly = TRUE)) {
  wide_plot_data <- predictions |>
    dplyr::mutate(match = paste(home_team, "vs", away_team)) |>
    dplyr::arrange(date, source_match_id) |>
    dplyr::slice_head(n = 12) |>
    dplyr::select(match, pred_home_win_prob, pred_draw_prob, pred_away_win_prob)

  plot_data <- rbind(
    data.frame(match = wide_plot_data$match, outcome = "Home win", probability = wide_plot_data$pred_home_win_prob),
    data.frame(match = wide_plot_data$match, outcome = "Draw", probability = wide_plot_data$pred_draw_prob),
    data.frame(match = wide_plot_data$match, outcome = "Away win", probability = wide_plot_data$pred_away_win_prob)
  )
  plot_data$match <- factor(plot_data$match, levels = rev(unique(wide_plot_data$match)))

  ggplot2::ggplot(plot_data, ggplot2::aes(x = probability, y = match, fill = outcome)) +
    ggplot2::geom_col(position = "stack", width = 0.72) +
    ggplot2::scale_x_continuous(labels = function(x) paste0(round(100 * x), "%")) +
    ggplot2::scale_fill_manual(values = c("Home win" = "#0f6b68", "Draw" = "#d9a441", "Away win" = "#b85c5c")) +
    ggplot2::labs(
      title = "Fixture result probabilities",
      x = "Predicted probability",
      y = NULL,
      fill = NULL
    ) +
    ggplot2::theme_minimal(base_size = 12)
} else {
  plot.new()
  text(0.5, 0.5, "Probability plot is available after the scoring refresh.")
}

6 Interpretation

The fixture table is the modelâ€™s operational output:

  • The Poisson goal estimates are the main expected-goals baseline.
  • The ordinal model supplies win/draw/loss probabilities.
  • OLS remains a transparent benchmark, but count models are better suited for goals.
  • Weather is shown as context until enough historical weather coverage exists for coefficient estimation.

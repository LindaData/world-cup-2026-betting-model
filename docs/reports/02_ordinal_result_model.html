Three-outcome match probabilities

1 Purpose

This report documents the result-probability model.

The ordered outcome is:

\[ Y_{i,m}^{ord} \in \{\text{loss}, \text{draw}, \text{win}\} \]

with:

\[ \text{loss} < \text{draw} < \text{win} \]

This output supports betting-market comparison because it estimates match outcome probabilities.

2 Unit Of Observation

The data has one row per team-match.

If Argentina plays France, there are two rows:

  • Argentina result versus France.
  • France result versus Argentina.

4 Model

The model estimates the chance that a team result is at or below each ordered level:

\[ \text{logit}\left(P(\text{Result} \leq k)\right) = \theta_k - \text{Team Score} \]

where:

\[ k \in \{\text{loss}, \text{draw}\} \]

and:

\[ \text{Team Score} = \beta_1\text{Team Elo} + \beta_2\text{Opponent Elo} + \beta_3\text{Expected Result} + \beta_4\text{Listed Home} + \beta_5\text{Neutral Site} \]

10 Interpretation

The model produces:

\[ P(\text{loss}), \quad P(\text{draw}), \quad P(\text{win}) \]

These probabilities are the shape used for odds comparison.

Model checks tracked here:

  • Proportional-odds fit.
  • Paired team-match dependence.
  • Draw calibration.
  • Comparison against market-implied probabilities.

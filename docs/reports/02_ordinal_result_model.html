Baseline model for loss, draw, and win probabilities

1 Purpose

This report adds the first result-probability model.

The ordered outcome is:

\[ Y_{i,m}^{ord} \in \{\text{loss}, \text{draw}, \text{win}\} \]

with:

\[ \text{loss} < \text{draw} < \text{win} \]

This is closer to betting-market comparison because it estimates match outcome probabilities.

2 Unit Of Observation

The data has one row per team-match.

If Argentina plays France, there are two rows:

  • Argentina result versus France.
  • France result versus Argentina.

4 Model

The proportional-odds ordinal logistic model is:

\[ \text{logit}\left(P(Y_{i,m}^{ord} \leq k)\right) = \theta_k - X_{i,m}\beta \]

where:

\[ k \in \{\text{loss}, \text{draw}\} \]

and:

\[ X_{i,m}\beta = \beta_1 E_{i,m}^{pre} + \beta_2 E_{j,m}^{pre} + \beta_3 p_{i,m} + \beta_4 I(\text{listed home}_{i,m}) + \beta_5 I(\text{neutral site}_m) \]

10 Interpretation

The model produces:

\[ P(\text{loss}), \quad P(\text{draw}), \quad P(\text{win}) \]

This is the shape needed for future odds comparison.

Still, this is a baseline:

  • The proportional-odds assumption may be too restrictive.
  • Paired team-match rows are not independent.
  • Draws usually need richer features.
  • Market odds will be the key benchmark once added.

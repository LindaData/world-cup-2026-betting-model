Count regression for expected team goals

1 Purpose

The Poisson goals model predicts how many goals a team is expected to score.

This is a better statistical shape for goals than simple linear regression because goals are counts: 0, 1, 2, 3, and so on. A Poisson model keeps the expected value positive and models scoring as a count process.

2 Model

The model predicts expected goals, usually written as lambda:

\[ \lambda = E(\text{Goals}) \]

Instead of fitting goals directly, the model fits the log of expected goals:

\[ \log(\lambda) = \beta_0 + \beta_1\text{Team Elo} + \beta_2\text{Opponent Elo} + \beta_3\text{Expected Result} + \beta_4\text{Listed Home} + \beta_5\text{Neutral Site} \]

Plain English: the model estimates a positive expected-goals number from team strength, opponent strength, expected result, and match context.

3 Validation

Two checks are shown:

  • Random 80/20: a general benchmark where rows are randomly split into training and test sets.
  • Train through 2018, test 2019+: a harder forward-looking check that better resembles predicting future matches.

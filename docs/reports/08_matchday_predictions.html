9 Post-Match Accuracy

accuracy_badge <- function(correct) {
  value <- as.logical(correct)[[1]]
  if (is.na(value)) {
    return('<span class="accuracy-badge accuracy-pending">Pending</span>')
  }
  if (value) {
    '<span class="accuracy-badge accuracy-correct">Right</span>'
  } else {
    '<span class="accuracy-badge accuracy-wrong">Wrong</span>'
  }
}

goal_error_chip <- function(value) {
  numeric_value <- safe_number(value)
  if (is.na(numeric_value) || !is.finite(numeric_value)) {
    return('<span class="error-chip error-neutral">No goal model</span>')
  }
  class_name <- if (numeric_value <= 0.5) {
    "error-good"
  } else if (numeric_value <= 1.25) {
    "error-warn"
  } else {
    "error-bad"
  }
  paste0(
    '<span class="error-chip ', class_name, '">',
    display_number(numeric_value, 2),
    ' goal miss</span>'
  )
}

probability_line <- function(value) {
  numeric_value <- safe_number(value)
  if (is.na(numeric_value) || !is.finite(numeric_value)) {
    return("")
  }
  paste0('<small>Actual result prob: ', display_percent(numeric_value, 1), '</small>')
}

model_cell <- function(row, pick_col, correct_col, prob_col = NULL, error_col = NULL) {
  pick <- safe_text(row[[pick_col]], "No pick")
  pieces <- c(
    paste0('<strong>', escape_html(pick), '</strong>'),
    accuracy_badge(row[[correct_col]])
  )
  if (!is.null(prob_col)) {
    pieces <- c(pieces, probability_line(row[[prob_col]]))
  }
  if (!is.null(error_col)) {
    pieces <- c(pieces, goal_error_chip(row[[error_col]]))
  }
  paste0('<div class="accuracy-model-cell">', paste(pieces[nzchar(pieces)], collapse = ""), '</div>')
}

10 Automatic Refresh And Model Updating

The model is designed to update after results arrive. The practical rule is:

\[ D_t = D_{t-1} \cup \{\text{latest final score, cards, lineup, weather, and news signals}\} \]

\[ \hat{f}_t = \operatorname{train}(D_t), \qquad \hat{y}_{t+1} = \hat{f}_t(X_{t+1}) \]

In plain English: after a match finishes, the pipeline adds the newest match data, rebuilds the modeling tables, reruns diagnostics, refits the models, renders the site, and publishes the updated prediction board.

Model selection starts broad, then narrows:

  • Data-quality checks: missingness, duplicate fixtures, impossible scores, and source freshness.
  • Regression diagnostics: collinearity, autocorrelation, heteroskedasticity, residual patterns, overdispersion, and grouped match effects.
  • Baseline models: OLS for goals, Poisson for score distributions, ordinal logistic regression for win/draw/loss, and KNN for historical similarity.
  • Machine-learning challengers: random forest and XGBoost-style boosted trees for nonlinear effects and interactions.
  • Promotion rule: a model must improve out-of-sample accuracy, calibration, and interpretability enough to justify the added complexity.

The board uses three active layers:

  • Winner and outcome: an ensemble of the ordinal logistic result model and probabilities derived from the Poisson score grid.
  • Goals and likely score: the Poisson goals model produces expected goals; the score grid turns those expected goals into scoreline probabilities.
  • Totals: over 2.5 goals and both-teams-to-score probabilities come from the same Poisson score grid.

Cards and lineups are live-enrichment layers. They update when the football API returns player card history, event rows, or confirmed lineups for a fixture.

12 How To Read It

  • Predicted winner: highest ensemble probability. A draw remains possible for group-stage style matches.
  • Confidence: the model’s top probability bucket, not a guarantee.
  • Expected goals: model average score, not the most likely exact score.
  • Most likely score: the single scoreline with the highest probability in the Poisson grid.
  • O2.5: probability that total goals are 3 or more.
  • BTTS: probability that both teams score at least one goal.

parse_utc_time <- function(x) {
  if (length(x) == 0 || is.null(x)) {
    return(as.POSIXct(NA_real_, origin = "1970-01-01", tz = "UTC"))
  }
  if (inherits(x, "POSIXt")) {
    return(as.POSIXct(x, tz = "UTC"))
  }
  text <- as.character(x)
  text <- sub("Z$", "", text)
  text <- gsub("T", " ", text, fixed = TRUE)
  as.POSIXct(text, format = "%Y-%m-%d %H:%M:%S", tz = "UTC")
}

  kickoff <- parse_utc_time(kickoff_iso)

build_projected_bracket <- function() {
  if (!requireNamespace("jsonlite", quietly = TRUE)) {
    return(NULL)
  }

  predictions <- read_csv_if_exists(file.path(root, "data", "processed", "modeling", "world_cup_2026_fixture_predictions.csv"))
  squads <- read_csv_if_exists(file.path(root, "data", "processed", "public_csv", "dim_2026_world_cup_squad_players.csv"))

  if (nrow(predictions) == 0 || nrow(squads) == 0) {
    return(NULL)
  }

  team_groups <- squads |>
    dplyr::select(team, group_name) |>
    dplyr::distinct() |>
    dplyr::mutate(
      group = gsub("^Group\\s+", "", group_name),
      team_join_key = team_key(team)
    ) |>
    dplyr::select(team_join_key, group)

  predictions <- predictions |>
    dplyr::mutate(
      home_join_key = team_key(home_team),
      away_join_key = team_key(away_team)
    ) |>
    dplyr::left_join(team_groups, by = c("home_join_key" = "team_join_key")) |>
    dplyr::rename(home_group = group) |>
    dplyr::left_join(team_groups, by = c("away_join_key" = "team_join_key")) |>
    dplyr::rename(away_group = group) |>
    dplyr::mutate(
      group = dplyr::coalesce(home_group, away_group),
      completed = !is.na(home_score) & !is.na(away_score),
      home_points = dplyr::case_when(
        completed & home_score > away_score ~ 3,
        completed & home_score == away_score ~ 1,
        completed & home_score < away_score ~ 0,
        TRUE ~ 3 * pred_home_win_prob + pred_draw_prob
      ),
      away_points = dplyr::case_when(
        completed & away_score > home_score ~ 3,
        completed & away_score == home_score ~ 1,
        completed & away_score < home_score ~ 0,
        TRUE ~ 3 * pred_away_win_prob + pred_draw_prob
      ),
      home_goals_for = dplyr::if_else(completed, as.numeric(home_score), pred_home_goals_poisson),
      away_goals_for = dplyr::if_else(completed, as.numeric(away_score), pred_away_goals_poisson),
      home_goals_against = away_goals_for,
      away_goals_against = home_goals_for
    ) |>
    dplyr::filter(!is.na(group))

  home_rows <- data.frame(
    group = predictions$group,
    team = predictions$home_team,
    points = predictions$home_points,
    gf = predictions$home_goals_for,
    ga = predictions$home_goals_against,
    completed = predictions$completed,
    elo = predictions$home_latest_elo
  )
  away_rows <- data.frame(
    group = predictions$group,
    team = predictions$away_team,
    points = predictions$away_points,
    gf = predictions$away_goals_for,
    ga = predictions$away_goals_against,
    completed = predictions$completed,
    elo = predictions$away_latest_elo
  )

  standings <- rbind(home_rows, away_rows) |>
    dplyr::group_by(group, team) |>
    dplyr::summarise(
      matches = dplyr::n(),
      completed = sum(completed, na.rm = TRUE),
      points = sum(points, na.rm = TRUE),
      gf = sum(gf, na.rm = TRUE),
      ga = sum(ga, na.rm = TRUE),
      gd = gf - ga,
      elo = max(elo, na.rm = TRUE),
      .groups = "drop"
    ) |>
    dplyr::arrange(group, dplyr::desc(points), dplyr::desc(gd), dplyr::desc(gf), dplyr::desc(elo), team) |>
    dplyr::group_by(group) |>
    dplyr::mutate(position = dplyr::row_number()) |>
    dplyr::ungroup()

  standings$elo[is.infinite(standings$elo)] <- NA_real_

  third_pool <- standings |>
    dplyr::filter(position == 3) |>
    dplyr::arrange(dplyr::desc(points), dplyr::desc(gd), dplyr::desc(gf), dplyr::desc(elo), team) |>
    dplyr::mutate(third_rank = dplyr::row_number(), qualifies = third_rank <= 8)

  slot_pairs <- data.frame(
    match_id = seq_len(16),
    slot_a = c("A2", "F1", "E1", "I1", "K2", "H1", "D1", "G1", "C1", "E2", "A1", "L1", "J1", "D2", "B1", "K1"),
    slot_b = c("B2", "C2", "ABCDF3", "CDFGH3", "L2", "J2", "BEFIJ3", "AEHIJ3", "F2", "I2", "CEFHI3", "EHIJK3", "H2", "G2", "EFGIJ3", "DEIJL3")
  )

  used_third_teams <- character()
  resolve_slot <- function(slot) {
    if (grepl("^[A-L][12]$", slot)) {
      group <- substr(slot, 1, 1)
      position <- as.integer(substr(slot, 2, 2))
      row <- standings[standings$group == group & standings$position == position, ]
      return(as_plain_record(row))
    }

    if (grepl("3$", slot)) {
      candidate_groups <- strsplit(sub("3$", "", slot), "")[[1]]
      row <- third_pool |>
        dplyr::filter(group %in% candidate_groups, !(team %in% used_third_teams), qualifies) |>
        dplyr::arrange(third_rank) |>
        dplyr::slice_head(n = 1)
      if (nrow(row) > 0) {
        used_third_teams <<- c(used_third_teams, row$team[[1]])
      }
      return(as_plain_record(row))
    }

    NULL
  }

  round32 <- lapply(seq_len(nrow(slot_pairs)), function(i) {
    row <- slot_pairs[i, ]
    list(
      match = as.integer(row$match_id),
      slotA = as.character(row$slot_a),
      slotB = as.character(row$slot_b),
      teamA = resolve_slot(as.character(row$slot_a)),
      teamB = resolve_slot(as.character(row$slot_b))
    )
  })

  group_summary <- standings |>
    dplyr::filter(position <= 3) |>
    dplyr::mutate(
      seed = paste0(group, position),
      points = round(points, 2),
      gd = round(gd, 2),
      gf = round(gf, 2),
      elo = round(elo, 0)
    ) |>
    dplyr::select(group, seed, team, position, points, gd, gf, elo) |>
    as.data.frame()

  third_summary <- third_pool |>
    dplyr::mutate(
      seed = paste0(group, "3"),
      points = round(points, 2),
      gd = round(gd, 2),
      gf = round(gf, 2),
      elo = round(elo, 0)
    ) |>
    dplyr::select(third_rank, qualifies, group, seed, team, points, gd, gf, elo) |>
    as.data.frame()

  list(
    round32 = round32,
    groupSeeds = group_summary,
    thirdWatch = third_summary,
    notes = list(
      seeding = "Projected group tables combine completed results with model expected points for remaining matches.",
      knockout = "Knockout advancement uses the current Elo-style strength signal until knockout match-specific data exists."
    )
  )
}

  function cloneTeam(team) {
    return team ? Object.assign({}, team) : null;
  }

  function cloneMatch(match) {
    return {
      slotA: match.slotA,
      slotB: match.slotB,
      teamA: cloneTeam(match.teamA),
      teamB: cloneTeam(match.teamB)
    };
  }

  function teamStrength(team) {
    const elo = team && Number(team.elo);
    return Number.isFinite(elo) ? elo : 1500;
  }

  function advanceProbability(teamA, teamB) {
    const diff = teamStrength(teamA) - teamStrength(teamB);
    return 1 / (1 + Math.pow(10, -diff / 400));
  }

  function formatPct(value) {
    if (!Number.isFinite(value)) {
      return "";
    }
    return Math.round(value * 100) + "%";
  }

  function modelWinner(match) {
    if (!match.teamA && !match.teamB) {
      return null;
    }
    if (match.teamA && !match.teamB) {
      return match.teamA;
    }
    if (!match.teamA && match.teamB) {
      return match.teamB;
    }
    return advanceProbability(match.teamA, match.teamB) >= 0.5 ? match.teamA : match.teamB;
  }

  function clearLaterPicks(roundIndex) {
    Object.keys(picks).forEach(function (key) {
      const keyRound = Number(key.split("-")[0]);
      if (keyRound > roundIndex) {
        delete picks[key];
      }
    });
  }

  function buildRounds() {
    const rounds = [data.round32.map(cloneMatch)];

    for (let roundIndex = 0; roundIndex < 4; roundIndex += 1) {
      const currentRound = rounds[roundIndex];
      const nextRound = Array.from({ length: Math.ceil(currentRound.length / 2) }, function (_, index) {
        return { slotA: "W" + (roundIndex + 1) + "." + (2 * index + 1), slotB: "W" + (roundIndex + 1) + "." + (2 * index + 2), teamA: null, teamB: null };
      });

      currentRound.forEach(function (match, matchIndex) {
        const key = roundIndex + "-" + matchIndex;
        const validNames = [match.teamA && match.teamA.name, match.teamB && match.teamB.name].filter(Boolean);
        if (picks[key] && !validNames.includes(picks[key])) {
          delete picks[key];
        }
        const pickedName = picks[key];
        const winner = pickedName
          ? validNames.includes(pickedName)
            ? (match.teamA && match.teamA.name === pickedName ? match.teamA : match.teamB)
            : modelWinner(match)
          : modelWinner(match);
        const nextMatch = nextRound[Math.floor(matchIndex / 2)];
        if (matchIndex % 2 === 0) {
          nextMatch.teamA = cloneTeam(winner);
        } else {
          nextMatch.teamB = cloneTeam(winner);
        }
      });

      rounds.push(nextRound);
    }

## Automatic Refresh And Model Updating

The model is designed to update after results arrive. The practical rule is:

$$
D_t = D_{t-1} \cup \{\text{latest final score, cards, lineup, weather, and news signals}\}
$$

$$
\hat{f}_t = \operatorname{train}(D_t), \qquad \hat{y}_{t+1} = \hat{f}_t(X_{t+1})
$$

In plain English: after a match finishes, the pipeline adds the newest match data, rebuilds the modeling tables, reruns diagnostics, refits the models, renders the site, and publishes the updated prediction board.

Model selection starts broad, then narrows:

- **Data-quality checks:** missingness, duplicate fixtures, impossible scores, and source freshness.
- **Regression diagnostics:** collinearity, autocorrelation, heteroskedasticity, residual patterns, overdispersion, and grouped match effects.
- **Baseline models:** OLS for goals, Poisson for score distributions, ordinal logistic regression for win/draw/loss, and KNN for historical similarity.
- **Machine-learning challengers:** random forest and XGBoost-style boosted trees for nonlinear effects and interactions.
- **Promotion rule:** a model must improve out-of-sample accuracy, calibration, and interpretability enough to justify the added complexity.

The board uses three active layers:

- **Winner and outcome:** an ensemble of the ordinal logistic result model and probabilities derived from the Poisson score grid.
- **Goals and likely score:** the Poisson goals model produces expected goals; the score grid turns those expected goals into scoreline probabilities.
- **Totals:** over 2.5 goals and both-teams-to-score probabilities come from the same Poisson score grid.

Cards and lineups are live-enrichment layers. They update when the football API returns player card history, event rows, or confirmed lineups for a fixture.

## How To Read It

- **Predicted winner:** highest ensemble probability. A draw remains possible for group-stage style matches.
- **Confidence:** the model's top probability bucket, not a guarantee.
- **Expected goals:** model average score, not the most likely exact score.
- **Most likely score:** the single scoreline with the highest probability in the Poisson grid.
- **O2.5:** probability that total goals are 3 or more.
- **BTTS:** probability that both teams score at least one goal.
